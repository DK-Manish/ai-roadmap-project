# Expense Tracker (CLI)

A command-line Python application to track and manage daily expenses.

## Features

* Add expenses (category + amount)
* View all expenses
* Calculate total spending
* Find highest expense
* Filter expenses by category
* Group expenses by category (summary)

## Tech Used

* Python
* Lists & Dictionaries
* Functions & Loops

## How to Run

```bash
python main.py
```

## Menu Options

```
1. Add Expense
2. Show Expenses
3. Total Spent
4. Highest Expense
5. Filter by Category
6. Grouped Summary
7. Exit
```

## Example

```
Enter category: Food
Enter amount: 100

Enter category: Travel
Enter amount: 200

--- Expense Summary ---
Food - 100
Travel - 200
```

# Student Report Card

A simple CLI-based Python application to generate report cards for students.

## Features

* Input student ID, name, and subject marks
* Calculates average marks
* Assigns grade based on average
* Supports multiple students
* Displays formatted report cards

## Tech Used

* Python
* Lists & Dictionaries
* Functions & Loops

## How to Run

```bash
python main.py
```

## Example

```
Enter number of students: 1

Enter student details:
Enter Student ID: 1
Enter Name: Manish
Enter marks for Maths: 80
Enter marks for Science: 90
Enter marks for English: 85

--- Report Card ---
ID: 1
Name: Manish
Marks: [80, 90, 85]
Average: 85.0
Grade: B
```
